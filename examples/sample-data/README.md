# Sample Data Files

This directory contains sample data files that can be used for testing and demonstration purposes with NocoDB.

## Files

### random_employee_data.xlsx
- **Description**: A sample employee dataset with 10 records
- **Columns**: Employee ID, Name, Age, Department, Salary, City, Join Date, Performance Score, Employment Status, Years of Experience
- **Use Case**: Perfect for testing data import functionality, creating dashboards, and demonstrating NocoDB features
- **Data Type**: Randomly generated Chinese employee data with realistic values

## How to Use

1. **Import to NocoDB**:
   - Open your NocoDB instance
   - Create a new table or base
   - Use the import feature to upload these Excel files
   - Map the columns appropriately

2. **Testing Features**:
   - Use this data to test filtering, sorting, and grouping
   - Create forms and views
   - Test API endpoints
   - Practice with different field types

## Data Generation

The sample data was generated using Python with the following characteristics:
- **Names**: Random Chinese names
- **Ages**: 22-60 years old
- **Departments**: Sales, Technology, HR, Finance, Marketing
- **Salaries**: 5,000-25,000 RMB
- **Cities**: Major Chinese cities (Beijing, Shanghai, Guangzhou, etc.)
- **Join Dates**: Random dates within the past year
- **Performance Scores**: 3.0-5.0 rating scale
- **Employment Status**: Active/Inactive employees
- **Experience**: 1-15 years of work experience

This data is completely fictional and generated for demonstration purposes only.
